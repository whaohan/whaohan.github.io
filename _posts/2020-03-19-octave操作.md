日常备忘

#### 矩阵生成

```Matlab
    eye(n)     % unit matrix

    ones(m,n)   % all one matrix

    zeros(m,n)   % all zero matrix

    inf(m,n)     % all infinte matrix

    rand(m,n)     % random number matrix

    a + sqrt(b) * randn(m,n)   % mean=a & variance=b

    []          % empty matrix
```

#### 存储数据
